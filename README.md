## Como este repositório está organizado

Cada questão contém:
- `prompt.md` — o prompt exato utilizado
- `modelo.md` — qual modelo foi usado e justificativa da escolha
- `output.*` — o output real gerado pelo modelo
- `justificativa.md` — análise de como os elementos R, T, F aparecem no prompt
